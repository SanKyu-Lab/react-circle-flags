## Import individual flags

Import flags from the main entry — **tree-shaking automatically** ensures you only bundle what you use:

```tsx
import { FlagUs, FlagCn, FlagGb } from '@sankyu/react-circle-flags'

export default function Header() {
  return (
    <div className="flex items-center gap-3">
      <FlagUs width={40} height={40} />
      <FlagCn width={40} height={40} />
      <FlagGb width={40} height={40} />
    </div>
  )
}
```

:::tip[Tree-shaking Optimization]
Modern bundlers automatically tree-shake unused flags. Importing 3 flags only adds **~2.8 KB** to your bundle, not the entire library.

No special configuration needed — it just works!
:::

## Naming convention

All flag components follow a consistent naming pattern:

<CardGrid>
  <Card title="Pattern" icon="star">
    Components use the format **`Flag{ISO}`** where ISO is the uppercase country code.
  </Card>
  <Card title="Examples" icon="list-format">
    - `FlagUs` → United States - `FlagCn` → China - `FlagGb` → Great Britain - `FlagJp` → Japan
  </Card>
</CardGrid>
